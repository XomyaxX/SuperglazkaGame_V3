## 🚨 Критично — стоит сделать в первую очередь

### 1. SEO и соцсети
| Проблема | Почему важно | Что сделать | Статус |
|----------|--------------|-------------|--------|
| Нет Open Graph / Twitter Card | При шаринге в соцсети ссылка выглядит как «голый» URL | Добавить в `<head>`: `og:title`, `og:description`, `og:image`, `og:url`, `twitter:card` (на обе страницы: `index.html` и `app.html`) | ✅ |
| Нет favicon | Во вкладке браузера — пустая иконка | Создать `favicon.ico` / `favicon.svg` и подключить | ✅ `favicon.svg` + PNG fallback |
| Нет `robots.txt` | Поисковики не знают, что индексировать | Создать `robots.txt` с `Allow: /` и указанием sitemap | ✅ |
| Нет `sitemap.xml` | Поисковики медленнее находят страницы | Сгенерировать `sitemap.xml` со всеми страницами (`index.html`, `app.html`) | ✅ |
| Нет canonical URL | Возможны дубли страниц в поиске | Добавить `<link rel="canonical">` на каждую страницу | ✅ |

### 2. PWA — сайт как «приложение»
| Задача | Статус |
|--------|--------|
| Создать `manifest.json` (название, иконки, цвета, `display: standalone`) | ✅ |
| Добавить Service Worker для **кеширования** — cache-first, офлайн-работа | ✅ |
| Добавить иконки в разных размерах (192×192, 512×512) для установки на телефон | ✅ `icons/icon-192x192.png`, `icons/icon-512x512.png`, `icons/icon-180x180.png` |

### 3. Доступность (Accessibility)
| Задача | Статус | Примечание |
|--------|--------|------------|
| `aria-label` на кнопки с иконками/эмодзи | ✅ | `profile-btn`, `cinema-toggle-btn`, `settingsMenuBtn`, `back-btn`, `nav-arrow-up/down`, `bs-narrator-toggle` |
| `role="button"` + `tabindex="0"` для кастомных элементов | ✅ | `bs-drag-handle` |
| `aria-live="polite"` для динамических текстов | ✅ | `subtitle-overlay`, `bs-narrator-full`, `bs-subtitle` |
