# Фоновая музыка по настроениям

Каждая папка — отдельное настроение (mood). Сайт автоматически выбирает папку на основу контента кадра (ключевые слова, цвета, сцена).

## Структура
```
moods/
  cosmic/         — Космос, чудо, планеты, звёзды
  joyful/         — Радость, дружелюбие, уют, веселье
  tension/        — Тьма, угроза, страх, тревога
  peaceful/       — Покой, гармония, природа, водопад
  magical/        — Ритуал, магия, заклинание, таинственность
  triumphant/     — Победа, трансформация, героизм, ликование
  warm/           — Тепло, дом, надежда, уют, вечер
  mystery/        — Тайна, неизвестность, вопросы, поиск
```

## Формат файлов
- MP3, 128–192 kbps
- Бесшовный loop (если возможно) или естественное начало/конец
- Без слов (инструментал)
- Длительность: 30–120 секунд (сайт зацикливает)

## Именование
`track01.mp3`, `track02.mp3` и т.д. Сайт выбирает случайный трек из папки.

## Примеры для каждого mood

### cosmic
- Эфирный эмбиент с синтезаторными пэдами
- Лёгкое мерцание, космические звуки
- Медленный темп, без ударных

### joyful
- Лёгкая фольк-музыка с флейтой/скрипкой
- Приподнятое настроение, умеренный темп
- Можно с лёгкими ударными

### tension
- Струнные стаккато, низкие басы
- Нарастающая динамика
- Минималистично, без мелодии

### peaceful
- Фортепиано или арфа
- Птички, шум воды (если Foley)
- Очень медленно, расслабляюще

### magical
- Колокольчики, струнные пэды
- Лёгкая перкуссия (треугольник)
- Сказочное, мистическое

### triumphant
- Медные духовые, литавры
- Героический оркестр
- Уверенно, величественно

### warm
- Акустическая гитара, тёплые пэды
- Вечерний уют
- Медленно, ностальгично

### mystery
- Фортепиано в миноре
- Лёгкая неопределённость
- Интрига, любопытство
