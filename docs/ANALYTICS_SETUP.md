## 1. Google Analytics 4 (GA4)

### Шаг 1. Создать аккаунт
1. Перейди на [analytics.google.com](https://analytics.google.com)
2. Нажми **"Начать бесплатно"** (или войди в свой Google-аккаунт)
3. Нажми **"Создать аккаунт"**

### Шаг 2. Настроить ресурс
1. **Название аккаунта**: `Superglazka`
2. **Название ресурса**: `Superglazka Website`
3. **Часовой пояс**: `Москва (UTC+3)`
4. **Валюта**: `Российский рубль`
5. Нажми **"Далее"**

### Шаг 3. Настроить бизнес
1. **Отрасль**: `Искусство и развлечения` → `Онлайн-игры` (или `Образование`)
2. **Размер бизнеса**: `Малый`
3. Нажми **"Создать"**

### Шаг 4. Получить ID потока данных
1. Выбери **"Веб"**
2. **URL сайта**: `https://vidial-media.ru`
3. **Название потока**: `Superglazka Web`
4. Нажми **"Создать поток"**
5. Скопируй **"Идентификатор потока данных"** — он выглядит как `G-XXXXXXXXXX`

### Шаг 5. Получить Measurement Protocol API Secret (опционально, для серверных событий)
1. В настройках потока → **"Measurement Protocol"**
2. Создай секрет и скопируй его

## 2. Яндекс.Метрика

### Шаг 1. Создать счётчик
1. Перейди на [metrika.yandex.ru](https://metrika.yandex.ru)
2. Войди в свой Яндекс-аккаунт
3. Нажми **"Добавить счётчик"**

### Шаг 2. Настроить счётчик
1. **Название**: `Суперглазка`
2. **Адрес сайта**: `https://vidial-media.ru`
3. **Часовой пояс**: `Москва`
4. **Валюта**: `Российский рубль`
5. Включи **"Вебвизор, карта скроллинга, аналитика форм"**
6. Включи **"Отслеживание хеша в адресной строке"** (важно для SPA)
7. Нажми **"Создать счётчик"**

### Шаг 3. Получить номер счётчика
1. После создания скопируй номер — он выглядит как `12345678`
2. Он отображается в шапке страницы счётчика

## 4. Что будет отслеживаться

### GA4 события (автоматические + кастомные)
- page_view — просмотр страницы
- session_start — начало сессии
- first_visit — первый визит
- click — клики по кнопкам
- scroll — глубина прокрутки
- Кастомные: `start_game`, `complete_chapter`, `play_minigame`

### Яндекс.Метрика (автоматические + цели)
- Просмотры страниц
- Время на сайте
- Глубина просмотра
- Вебвизор (запись действий пользователя)
- Цели: `START_GAME`, `COMPLETE_CHAPTER`, `REGISTER`
