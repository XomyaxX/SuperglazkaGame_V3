# 📥 INCOMING — Папка для новых элементов

Сюда складывай любые новые файлы (кадры, аудио, видео, персонажей), которые нужно добавить в проект.

## 📋 Как пользоваться

1. **Создай подпапку** с названием эпизода или типа контента, например:
   - `incoming/episode-04/` — для нового эпизода
   - `incoming/new-character/` — для нового персонажа
   - `incoming/game-assets/` — для ресурсов игр

2. **Положи файлы** внутрь подпапки. Используй понятные имена:
   - `frame-01.png` (изображение кадра)
   - `narration.mp3` (озвучка рассказчика)
   - `dialogue.mp3` (озвучка диалога)
   - `video.mp4` (видеовставка)

3. **Напиши инструкцию** в файле `incoming/episode-XX/README.txt`:
   ```
   Эпизод 4 — «Название эпизода»
   
   Кадры:
   - frame-01.png → Кадр 1: Описание сцены
   - frame-02.png → Кадр 2: Описание сцены
   
   Аудио:
   - narration-01.mp3 → Озвучка кадра 1
   
   Видео:
   - video-01.mp4 → Видео кадра 1
   
   Диалоги (если есть):
   - dialogue-01.mp3 → Диалог в кадре 2
   
   Мини-игры (если есть):
   - blink на кадре 3
   - tracker на кадре 5
   ```

4. **Сообщи мне** — я сам рассортирую файлы по правильным папкам, обновлю пути в коде и настрою эпизод.

## 🎮 Типы мини-игр

- `blink` — Моргай-зарядка (тренировка глаз)
- `tracker` — Следи за шариком
- `runner` — Погоня (запускается отдельно)
- `gym` — Бой с Ленивусом (запускается отдельно)

## 💡 Советы

- Не используй пробелы и кириллицу в именах файлов — только латиница, цифры, дефисы и подчёркивания.
- Форматы: `.png` или `.jpg` для картинок, `.mp3` для аудио, `.mp4` для видео.
- Разрешение кадров: рекомендуется 1920×1080 или вертикальное 1080×1920.
